xx-net翻墙工具配置

Posted by Juan on 2016-11-10

mac平台下配置xx-net

简介

xx-net是一款集成goagent，比goagent配置更加简单，访问速度更快的翻墙工具，特点：

  • 集成搜索Google IP
  • 基于web的配置页面，人性化交互
  • 内置了公共appid,安装即可翻墙上网（建议创建自己的appid)

下载

选择稳定版本下载
https://github.com/XX-net/XX-Net/blob/master/code/default/download.md

安装

  1. 解压下载好的压缩包，点击start运行，启动xx-net
  2. 打开http://127.0.0.1:8085/查看GAEProxy状态信息

手动安装CA认证

  1. Finder -> 应用程序 -> 实用工具文件夹 -> 钥匙串访问
  2. 文件 -> 导入项目 -> /XX-Net-3.x.x/data/gae_proxy/CA.crt

配置http代理（推荐Chrome + SwitchyOmega）

直接参考windows下使用浏览器设置xx-net设置代理方案二章节

创建部署自己的appid

虽然系统内置了公共appid, 还是建议部署自己的appid，公共appid限制看视频。
